3. 블로그 작성법

1단계 — 기본 정보 입력

1
좌측 메뉴에서 [블로그] 클릭
2
[새 글 작성] 탭이 기본 선택됩니다
3
제목(H1) 입력 — SEO 키워드를 포함하세요
좋은 예: 임플란트 수술 후 주의사항 7가지 — 현진호 원장 가이드
나쁜 예: 수술 후 주의사항
4
카테고리 선택: 일반 / 임플란트 / 심미치료 / 신경치료 / 치과상식
5
담당 원장 선택 (선택사항)
6
메타 설명 입력 (선택) — 네이버/구글 검색결과에 표시되는 요약문. 160자 이내 권장

2단계 — 본문 작성 (SEO 블록 에디터)

본문은 블록 단위로 작성합니다. 상단 툴바의 버튼으로 블록을 추가합니다:

버튼기능사용 팁
H2소제목글의 큰 주제 구분 (검색엔진이 중요하게 봄)
H3중제목H2 아래 세부 주제
본문일반 텍스트설명 문단
목록리스트주의사항, 장점 등 나열
인용인용문강조 문구
이미지사진 삽입클릭하면 파일 선택 / 드래그 가능
B / I굵게 / 기울임텍스트 선택 후 클릭 (Ctrl+B / Ctrl+I)
링크하이퍼링크텍스트 선택 후 URL 입력
미리보기결과 미리보기실제 사이트 보이는 모습 확인
블록 위에 마우스를 올리면 오른쪽 상단에 이동/삭제 버튼이 나타납니다. 블록 사이에 마우스를 올리면 [+] 버튼으로 중간에 블록 추가도 가능합니다.

3단계 — 썸네일 이미지 (선택)

하단 이미지 업로드 영역에 대표 이미지를 드래그 & 드롭하세요. 비워두면 본문 첫 번째 이미지가 자동으로 썸네일이 됩니다.

4단계 — 등록

!
[게시글 등록] 클릭 → 홈페이지 블로그에 즉시 반영

기존 글 수정 / 삭제

  1. [게시글 목록] 탭 클릭
  2. 우측 [수정] → 에디터에 내용 불러옴 → 수정 후 [수정 저장]
  3. [삭제] → 확인 팝업 → 삭제 완료

SEO 체크리스트

에디터 하단에 SEO 힌트가 자동 표시됩니다:

  • 제목 길이 (30~70자 권장)
  • H2 소제목 포함 여부
  • 본문 길이 (300자 이상 권장)
  • 이미지 포함 여부
  • 메타 설명 입력 여부

4. 비포&애프터 작성법

1단계 — 기본 정보

1
좌측 메뉴에서 [비포&애프터] 클릭
2
케이스 제목 입력 (예: 상악 좌측 제1대구치 임플란트)
3
카테고리 선택: 임플란트 / 심미치료 / 신경치료 / 교정 / 기타
4
담당 원장 선택 (선택사항)
5
설명 입력 (선택) — 치료 과정, 기간 등

2단계 — 사진 업로드

총 4개의 슬롯이 있습니다. 각 슬롯을 클릭하면 파일 선택 창이 뜹니다.

슬롯설명
구내사진 BEFORE치료 전 구강 내 사진
구내사진 AFTER치료 후 구강 내 사진
파노라마 BEFORE치료 전 파노라마 X-ray
파노라마 AFTER치료 후 파노라마 X-ray
환자 개인정보가 노출되지 않도록 사진에서 이름, 차트번호 등을 반드시 가려주세요.
모든 슬롯을 채울 필요는 없습니다 — 있는 사진만 올리면 됩니다. 사진은 R2 클라우드 스토리지에 안전하게 저장됩니다.

3단계 — 등록

!
[케이스 등록] 클릭 → 홈페이지에 즉시 반영

기존 케이스 수정 / 삭제

  1. [케이스 목록] 탭 클릭
  2. [수정] → 내용/사진 변경 → [수정 저장]
  3. [삭제] → R2에 저장된 사진도 자동 삭제

5. 공지사항 작성법

1단계 — 내용 작성

1
좌측 메뉴에서 [공지사항] 클릭
2
제목 입력 (예: 2026년 5월 연휴 휴진 안내)
3
내용 입력 — 일반 텍스트 에디터 (줄바꿈 가능)

2단계 — 사진 첨부 (선택)

이미지 업로드 영역에 파일을 드래그 & 드롭하거나 클릭해서 선택하세요. 여러 장 동시 업로드 가능합니다.

3단계 — 고정 설정 (선택)

[상단 고정]을 체크하면 공지 목록 최상단에 항상 표시됩니다. 진료시간 안내, 주차 안내 같은 중요 공지에 사용하세요.

4단계 — 등록

!
[공지 등록] 클릭 → 홈페이지에 즉시 반영

6. 조회수 확인

모든 콘텐츠의 조회수는 각 섹션의 목록 탭에서 확인할 수 있습니다.

콘텐츠확인 경로표시 형태
블로그블로그 → 게시글 목록👁 숫자
비포&애프터비포&애프터 → 케이스 목록👁 숫자
공지사항공지사항 → 공지 목록👁 숫자
백과사전백과사전 → 용어 목록조회 숫자
조회수 기준: 사이트 방문자가 해당 게시물을 클릭하여 상세 페이지를 볼 때마다 1씩 증가합니다. 관리자가 어드민에서 조회할 때는 증가하지 않습니다.

7. 빠른 가이드 (3줄 요약)

📝 블로그 올리기 (3분)

[블로그] → 제목 입력 → 카테고리 선택 → 본문 작성 → [게시글 등록]

📸 비포&애프터 올리기 (2분)

[비포&애프터] → 제목 + 카테고리 → 사진 업로드 → [케이스 등록]

📢 공지사항 올리기 (1분)

[공지사항] → 제목 + 내용 → (고정 체크) → [공지 등록]

8. 문제 해결 FAQ

Q. 로그인이 안 돼요

비밀번호를 다시 확인해주세요. 변경한 적 있으면 변경된 비밀번호를 입력하세요. 브라우저 캐시를 삭제하고 재시도해보세요.

Q. 이미지가 업로드 안 돼요

  • 파일 크기: 10MB 이하 권장
  • 파일 형식: JPG, PNG, WebP 지원
  • 인터넷 연결 상태를 확인하세요

Q. 작성한 글이 사이트에 안 보여요

  • 목록에서 해당 글의 상태가 [공개]인지 확인
  • 브라우저에서 Ctrl + Shift + R (강력 새로고침) 시도
  • 등록 후 1~2분 정도 캐시 반영 시간이 있을 수 있음

Q. 비밀번호를 잊어버렸어요

개발팀에 연락하여 비밀번호 초기화를 요청해주세요.
