## 3. 블로그 작성법

### 새 글 작성 순서

**1단계 — 기본 정보 입력**
1. 좌측 메뉴에서 [블로그] 클릭
2. [새 글 작성] 탭이 기본으로 선택되어 있습니다
3. **제목(H1)** 입력 — SEO 키워드를 포함하세요
   - 좋은 예: `임플란트 수술 후 주의사항 7가지 — 현진호 원장 가이드`
   - 나쁜 예: `수술 후 주의사항`
4. **카테고리** 선택: 일반 / 임플란트 / 심미치료 / 신경치료 / 치과상식
5. **담당 원장** 선택 (선택사항): 드롭다운에서 원장 선택
6. **메타 설명** 입력 (선택) — 네이버/구글 검색 결과에 표시될 요약문 (160자 이내)

**2단계 — 본문 작성 (SEO 블록 에디터)**

본문은 **블록 단위**로 작성합니다. 상단 툴바의 버튼으로 블록을 추가합니다:

| 버튼 | 기능 | 사용 팁 |
|---|---|---|
| **H2** | 소제목 | 글의 큰 주제별 구분 (검색엔진이 중요하게 봄) |
| **H3** | 중제목 | H2 아래 세부 주제 |
| **본문** | 일반 텍스트 | 설명 문단 |
| **목록** | 리스트 | 주의사항, 장점 등 나열 |
| **인용** | 인용문 | 강조하고 싶은 문구 |
| **이미지** | 사진 삽입 | 클릭하면 파일 선택/드래그 가능 |
| **B** | 굵게 | 텍스트 선택 후 클릭 (Ctrl+B) |
| **I** | 기울임 | 텍스트 선택 후 클릭 (Ctrl+I) |
| **링크** | 하이퍼링크 | 텍스트 선택 후 클릭, URL 입력 |
| **미리보기** | 작성 결과 미리보기 | 실제 사이트에 보이는 모습 확인 |

> **TIP**: 블록 위에 마우스를 올리면 오른쪽 상단에 이동/삭제 버튼이 나타납니다.
> 블록 사이에 마우스를 올리면 [+] 버튼으로 중간에 블록 추가 가능합니다.

**3단계 — 썸네일 이미지 (선택)**
- 하단의 이미지 업로드 영역에 대표 이미지를 드래그 & 드롭
- 비워두면 본문의 첫 번째 이미지가 자동으로 썸네일로 사용됩니다

**4단계 — 등록**
- [게시글 등록] 버튼 클릭
- "등록되었습니다" 알림이 나오면 완료
- 홈페이지 블로그 페이지에 **즉시 반영**됩니다

### 기존 글 수정/삭제
1. [게시글 목록] 탭 클릭
2. 게시글 우측의 [수정] 버튼 → 에디터에 내용 불러옴 → 수정 후 [수정 저장]
3. [삭제] 버튼 → 확인 팝업 → 삭제 완료

### 조회수 확인
- **게시글 목록**에서 각 글 제목 아래에 👁 아이콘과 함께 조회수가 표시됩니다
- 예: `임플란트 · 2026.04.09 · 👁 8 [공개]`

### SEO 체크리스트
에디터 하단에 SEO 힌트가 자동으로 표시됩니다:
- 제목 길이 (30~70자 권장)
- H2 소제목 포함 여부
- 본문 길이 (300자 이상 권장)
- 이미지 포함 여부
- 메타 설명 입력 여부

## 4. 비포&애프터 작성법

### 새 케이스 등록 순서

**1단계 — 기본 정보**
1. 좌측 메뉴에서 [비포&애프터] 클릭
2. [새 케이스 등록] 탭 선택
3. **케이스 제목** 입력
   - 좋은 예: `상악 좌측 제1대구치 임플란트`
   - 좋은 예: `상악 전치부 6본 포셀린 라미네이트`
4. **카테고리** 선택: 임플란트 / 심미치료 / 신경치료 / 교정 / 기타
5. **담당 원장** 선택 (선택사항)
6. **설명** 입력 (선택) — 치료 과정, 기간 등 간략 설명

**2단계 — 사진 업로드**

총 4장의 슬롯이 있습니다:

| 슬롯 | 설명 |
|---|---|
| **구내사진 BEFORE** | 치료 전 구강 내 사진 |
| **구내사진 AFTER** | 치료 후 구강 내 사진 |
| **파노라마 BEFORE** | 치료 전 파노라마 X-ray |
| **파노라마 AFTER** | 치료 후 파노라마 X-ray |

- 각 슬롯을 클릭하면 파일 선택 창이 뜹니다
- 사진을 선택하면 자동으로 업로드되고 미리보기가 표시됩니다
- 모든 슬롯을 채울 필요는 없습니다 — 있는 사진만 올리면 됩니다

> **중요**: 사진은 R2 클라우드 스토리지에 안전하게 저장됩니다.
> 환자 개인정보가 노출되지 않도록 사진에서 이름/번호 등을 가려주세요.

**3단계 — 등록**
- [케이스 등록] 버튼 클릭
- 홈페이지 비포&애프터 페이지에 **즉시 반영**

### 기존 케이스 수정/삭제
1. [케이스 목록] 탭 클릭
2. 목록에서 조회수(👁), 카테고리, 사진 수 확인 가능
3. [수정] → 내용 수정 → [수정 저장]
4. [삭제] → R2에 저장된 사진도 자동 삭제

### 조회수 확인
- 케이스 목록에서 각 항목에 👁 아이콘과 조회수가 표시됩니다
- 예: `임플란트 · 사진 4장 · 2026.04.09 · 👁 5 [공개]`

## 5. 공지사항 작성법

### 새 공지 작성 순서

**1단계 — 내용 작성**
1. 좌측 메뉴에서 [공지사항] 클릭
2. [새 공지 작성] 탭 선택
3. **제목** 입력
   - 예: `2026년 5월 연휴 휴진 안내`
   - 예: `[이벤트] 여름맞이 스케일링 할인`
4. **내용** 입력 — 일반 텍스트 에디터 (줄바꿈 가능)

**2단계 — 사진 첨부 (선택)**
- 이미지 업로드 영역에 파일을 드래그 & 드롭하거나 클릭해서 선택
- 여러 장 동시 업로드 가능
- 업로드된 사진은 공지 상세 페이지에 표시됩니다

**3단계 — 고정 설정 (선택)**
- **[상단 고정]** 체크 → 공지 목록 최상단에 항상 표시
- 진료시간 안내, 주차 안내 같은 중요 공지에 사용

**4단계 — 등록**
- [공지 등록] 버튼 클릭
- 홈페이지 공지사항 페이지에 **즉시 반영**

### 기존 공지 수정/삭제
1. [공지 목록] 탭 클릭
2. 목록에서 조회수, 고정/공개 상태 확인 가능
3. [수정] → 내용 수정 → [수정 저장]
4. [삭제] → 확인 후 삭제

### 조회수 확인
- 공지 목록에서 각 항목에 👁 아이콘과 조회수가 표시됩니다
- 예: `2026.04.09 · 👁 5 [공개] [고정]`

## 6. 조회수 확인 총정리

모든 콘텐츠의 조회수는 **각 섹션의 목록 탭**에서 확인할 수 있습니다.

| 콘텐츠 | 조회수 위치 | 표시 형태 |
|---|---|---|
| **블로그** | 블로그 → 게시글 목록 | 👁 숫자 |
| **비포&애프터** | 비포&애프터 → 케이스 목록 | 👁 숫자 |
| **공지사항** | 공지사항 → 공지 목록 | 👁 숫자 |
| **백과사전** | 백과사전 → 용어 목록 | 조회 숫자 |

> **조회수 기준**: 사이트 방문자가 해당 게시물을 클릭하여 상세 페이지를 볼 때마다 1씩 증가합니다.
> 관리자가 어드민에서 조회할 때는 증가하지 않습니다.

## 8. 문제 해결 (FAQ)

### Q. 로그인이 안 돼요
- 비밀번호 확인: `gaon2026!` (변경한 적 있으면 변경된 비밀번호)
- 브라우저 캐시 삭제 후 재시도

### Q. 이미지가 업로드 안 돼요
- 파일 크기 확인: 10MB 이하 권장
- 파일 형식 확인: JPG, PNG, WebP 지원
- 인터넷 연결 상태 확인

### Q. 작성한 글이 사이트에 안 보여요
- [게시글 목록]에서 해당 글의 상태가 [공개]인지 확인
- 브라우저에서 Ctrl+Shift+R (강력 새로고침) 시도
- 등록 후 약 1~2분 정도 캐시 반영 시간이 있을 수 있음

### Q. 비밀번호를 잊어버렸어요
- 개발팀에 연락하여 비밀번호 초기화 요청
